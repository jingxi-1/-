# 回忆树互动网页 🌳

## 项目简介
“回忆树”是一款记录和分享朋友回忆的互动网页应用。用户可以通过浇水、摇叶子、上传回忆等方式，与自己的旅行、生活和友情记忆进行互动。网页整体风格文艺、温暖、高级，提供沉浸式体验。

## 功能亮点
- **开场动画**：展示文艺文字，“世界这么大，这么多人，我们冥冥之中相遇……”
- **回忆树主页**：展示树干、枝叶和叶子，叶子绑定回忆（文字/图片/视频）
- **浇水功能**：点击浇水按钮，树叶生长、开花，寓意友情或回忆被滋养
- **摇一摇功能**：点击按钮或手机摇动，随机掉落一片叶子，显示一条回忆
- **上传回忆**：用户可上传新照片、文字或短视频，生成新的叶子
- **分享功能**：生成网页链接或二维码，好友可访问并互动

## 技术栈
- **开发工具**：Claude Code + VibeCoding
- **大模型**：Kimi Code API
- **前端**：HTML5, CSS3, JavaScript, SVG/Canvas
- **动画库**：GSAP
- **数据存储**：Supabase

## 使用方法
1. 打开 `index.html` 文件，即可查看网页原型
2. 点击“浇水”或“摇一摇”体验互动效果
3. 点击上传回忆按钮，可新增叶子和回忆内容
